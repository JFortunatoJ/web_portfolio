O jogo foi inicialmente desenvolvido para dispositivos móveis (Android e iOS), porém para facilitar a acessibilidade, decidimos portá-lo para Web, não sendo necessário realizar download direto, sendo possível jogá-lo tanto do navegador de um computador quanto de um celular.


O principal desafio foi otimizar o jogo para dispositivos móveis, porque até então o SDK Web GL da Unity não possui suporte para ser executado em celulares, como resultado, enquanto no navegador do computador o jogo executava com tranquilidade, em dispositivos móveis o framerate caía drasticamente, sendo praticamente injogável em alguns casos. Eu com os meus colegas então passamos alguns bons dias pesquisando, testando e gerando muitas builds, e como há relativamente pouco conteúdo na internet sobre o tema e essa foi basicamente nossa primeira experiência no desenvolvimento de um jogo para Web GL utilizando Unity e que precisaria funcionar em dispositivos móveis, fomos progredindo na base da tentativa e erro. Chegamos a cogitar em desenvolver o jogo utilizando outra engine como Play Canvas que é totalmente focada em jogos para web, porém o prazo de entrega não seria suficiente para aprendermos uma nova engine e desenvolver o jogo novamente nela.


Por fim conseguimos acertar na otimização do jogo e entregar dentro do prazo. Apesar das dificuldades, foi um ótimo aprendizado que nos ensinou muito sobre builds Web GL utilizando a Unity, as configurações necessárias e práticas de desenvolvimento e interação com os colegas que ajudam a agilizar o trabalho.

O login com o Facebook foi utilizado para salvar os dados do jogador e para que o mesmo tenha acesso ao Ranking Global de Jogadores.

O processo é dividido em duas etapas: na primeira o jogador realiza o login com o facebook no qual é adquirido o token de acesso, na segunda etapa é feita a autenticação com o backend do jogo utilizando o token do Facebook do jogador para identificá-lo e conseguir a sua lista de amigos que também jogam o jogo para popular o ranking de amigos.
